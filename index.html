HƯỚNG DẪN THAM GIA

Chỉ với 3 bước đơn giản để kết nối vào thế giới sinh tồn của chúng tôi

BƯỚC 1

Tải Launcher

Khuyên dùng Prism Launcher Cracked để dễ dàng quản lý phiên bản và cài đặt modpack. Nếu dùng official account có thể dùng Prism Launcher thường và modrinth.

Prism Launcher Cracked (Khuyên dùng) Prism Launcher (Chính thức) Modrinth App (Chính thức)
BƯỚC 2

Cài Đặt Modpack

Tải tệp modpack của server (định dạng .mrpack hoặc thư mục .zip). Trong Prism Launcher, chọn Add Instance > Import > Chọn tệp đã tải. Phiên bản 1.21.1 và Neoforge sẽ tự động thiết lập.

Tải File .mrpack (Nhanh) Tải File .zip (Dự phòng)
BƯỚC 3
